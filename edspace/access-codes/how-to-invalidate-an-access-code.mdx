---
title: How to invalidate an access code?
description: Invalidation of Access Codes in Memorang Hub.
---

To invalidate an Access Code:

1. Select an organization from the menu on the left.
2. Choose a project from the dashboard to access the services page.
3. **Navigate to Access Codes:** Go to EdSpace -> Access Codes.
4. **Invalidate Code:** Use the Order Actions section to invalidate the access code.
