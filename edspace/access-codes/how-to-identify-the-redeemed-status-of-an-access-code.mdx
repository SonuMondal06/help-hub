---
title: How to identify the redeemed status of an access code?
description: Checking Redeemed Status of Access Codes.
---

To check if an access code has been redeemed:

1. **Select an Organization:** Choose the organization from the selection menu on the left.
2. **Navigate to the Project:** Select a project from the dashboard to reach the services page.
3. **Navigate to Access Codes:** Go to EdSpace -> Access Codes.
4. **Check Redeemed Column:** Look at the ‘Redeemed’ column for the status of each access code.
