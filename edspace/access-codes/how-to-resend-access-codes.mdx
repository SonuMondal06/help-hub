---
title: How to resend access codes?
description: Resending Access Codes in Memorang Hub.
---

To resend access codes:

1. **Select an Organization:** Choose the organization from the left-hand menu.
2. **Navigate to the Project:** Select a project from the dashboard to go to the services page.
3. **Navigate to Access Codes:** Go to EdSpace -> Access Codes.
4. **Order Actions:** Use the Order Actions section to send instructions to a new or existing email.
