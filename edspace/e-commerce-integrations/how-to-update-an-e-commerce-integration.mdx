---
title: How to update an e-commerce integration?
description: Updating E-commerce Integrations in Memorang Hub.
---

To update an existing e-commerce integration in Memorang Memorang:

1. **Select the Record to Update:** Go to the "Update" tab on the E-commerce Integration Page.
2. **Modify External IDs:** Update the External Product ID and Plan ID fields as needed.
3. **Save Changes:** Click Update to apply the modifications.
