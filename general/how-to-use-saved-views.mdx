---
title: How to use saved views
description: Utilizing Saved Views in Memorang Hub.
---

Saved views allow you to apply predefined filters to your orders list in Memorang. Here's how to use them:

1. **Open Saved Views:** Go to EdSpace -> Orders and click the Views button above the table.
2. **Apply a Saved View:** Select a view from the list to apply its associated filters.


## Creating a New View

1. Apply your desired filters.
2. Click the Views button, then select 'Create View'.
3. Name the view in the dialog box and click Create.

## Deleting a View

1. Open the list of views by clicking the Views button.
2. Use the delete button next to each view and confirm deletion
